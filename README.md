# 🍕 Pizza Sales Analysis Using SQL

## 📌 Project Overview
This project analyzes pizza sales data using SQL to uncover business insights such as revenue trends, top-selling pizzas, category performance, and customer ordering patterns.

## 🎯 Business Problem
The pizza business wants to understand:
- Which pizzas generate the most revenue
- Which categories perform best
- Sales trends over time
- Top-selling pizza types
- Revenue contribution by category

## 🛠️ Tools Used
- PostgreSQL
- SQL
- pgAdmin 4
- GitHub

## 📊 Key Analysis Performed
- Total Orders
- Total Revenue
- Highest Priced Pizza
- Most Common Pizza Size
- Top 5 Most Ordered Pizza Types
- Revenue by Pizza Category
- Cumulative Revenue Analysis
- Top 3 Pizza Types by Revenue in Each Category
- Percentage Contribution of Each Category

## 🧠 SQL Concepts Used
- Joins
- Aggregate Functions
- GROUP BY
- ORDER BY
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- RANK()
